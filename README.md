## Mac, Debian, RPM installer

### Install (default)

curl -fsSL "https://raw.githubusercontent.com/lyraminds/xdr-agent-install/main/install.sh" | sudo bash

### Install with custom name

curl -fsSL "https://raw.githubusercontent.com/lyraminds/xdr-agent-install/main/install.sh" | sudo bash -s -- install my-server-1

### Re-enroll

curl -fsSL "https://raw.githubusercontent.com/lyraminds/xdr-agent-install/main/install.sh" | sudo bash -s -- enroll

### Remove

curl -fsSL "https://raw.githubusercontent.com/lyraminds/xdr-agent-install/main/install.sh" | sudo bash -s -- remove

### Status

curl -fsSL "https://raw.githubusercontent.com/lyraminds/xdr-agent-install/main/install.sh" | sudo bash -s -- status
